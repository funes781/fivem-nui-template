# SIMPLE NUI TEMPLATE

A high-performance boilerplate for FiveM Native User Interfaces (NUI) built with React 19, Vite, and TypeScript. This template is optimized for modern web standards within the CitizenFX framework.

---

## 1. Technical Stack

### Core Dependencies
* **React (v19.2.0):** UI library for building component-based interfaces.
* **React-DOM (v19.2.0):** Entry point for DOM-related rendering.

### Development Tools
* **Vite (v7.3.1):** Next-generation frontend tooling for fast builds and HMR.
* **TypeScript (~5.9.3):** Static type checking for robust code.
* **ESLint (v9.39.1):** Pluggable linting utility for maintaining code quality.
* **PostCSS & TailwindCSS:** (Configured for utility-first styling and class obfuscation).

---

## 2. Available Scripts

Execute these commands from the project root:

| Command | Description |
| :--- | :--- |
| `npm run dev` | Starts the local development server with Hot Module Replacement (HMR). |
| `npm run build` | Compiles TypeScript and builds the production-ready assets into the `/dist` folder. |
| `npm run lint` | Runs ESLint to identify and fix code style issues and potential bugs. |
| `npm run preview` | Locally previews the production build generated by the build script. |
| `npm run test:ui` | Executes Playwright responsiveness and visual regression tests. |
